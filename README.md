
# Mini CRM - MERN Full Scaffold (Starter)

This is a full MERN starter scaffold including:
- Backend: Node.js + Express + Mongoose (JWT auth, customers, leads)
- Frontend: React (Register, Login, Dashboard, Customers, Customer detail + Leads)
- Joi validation for requests
- Role-based middleware (admin|user)
- One Jest + Supertest test for auth
- Simple reporting chart (Chart.js)

Folders:
- backend/
- frontend/

See `RUN_LOCALLY.md` for step-by-step local run instructions.
